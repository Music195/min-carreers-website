32常用対数

基本事項

1. 常用対数（10 を底とする対数 \( \log_{10}N \)）
正の数 \( N \) は \( a \times 10^n \quad (1 \leq a < 10,\ n \text{ は整数}) \) で表され、
\( \log_{10}N = n + \log_{10}a \quad \text{（ただし } 0 \leq \log_{10}a < 1 \text{）} \)
2. 正の数の整数部分の桁数、小数桁位
  1. \( N \) の整数部分が \( k \) 桁 ⇔
    \( 10^{k-1} \leq N < 10^k \iff k - 1 \leq \log_{10}N < k \)
  2. \( N \) は小数第 \( k \) 位に初めて 0 でない数字が現れる ⇔
    \( \frac{1}{10^k} \leq N < \frac{1}{10^{k-1}} \iff -k \leq \log_{10}N < -k + 1 \)
